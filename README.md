# UVA Cyber & IP Guide

An interactive cybersecurity and intellectual property (IP) awareness website built for University of Virginia students. The goal of this project is to make cybersecurity concepts more accessible, engaging, and relevant to college students through simulations and real-world examples.

---

## Overview

A lot of cybersecurity resources are either too dense or not tailored toward students. This project focuses on simplifying key concepts like phishing, social engineering, and password security while also adding interactive elements so users can actually practice making decisions.

Instead of just reading information, users can:
- go through scenario-based simulations
- test their knowledge
- explore common cybersecurity risks students face

---

## Features

- Educational Pages
  - Phishing
  - Social Engineering
  - Password Attacks
  - IP Rights / Guidelines

- Interactive Simulation
  - BitLife / Oregon Trail–style decision-making game
  - Users go through real-life cybersecurity scenarios and choose actions
  - Live feedback based on choices the user makes

---

## Tech Stack

This project is built as a static frontend site:

- HTML
- CSS
- JavaScript

Project structure:
- index.html – homepage
- cyber.html – cybersecurity content
- ip.html – IP guidelines
- simulation.html – interactive scenario game
- css/ – styling
- js/ – scripts and logic

---

## How It Works

- Most of the interactivity is handled through JavaScript
- The simulation uses basic state logic to move between scenarios
- The quiz tracks answers and provides feedback in real time
- The site is deployed using GitHub Pages

## Purpose

This project was created as part of a first-year engineering foundations course at UVA. It aims to make cybersecurity education more interactive and relevant for students, especially engineering students who may underestimate these risks because they assume being tech-savvy makes them less vulnerable.
